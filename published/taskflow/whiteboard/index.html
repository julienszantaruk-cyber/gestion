Type d'élément

🏛 Whiteboard Collaboratif

Tableau blanc connecté à Grist pour la facilitation d'ateliers.

Raccourcis

  • V Sélection   S Sticky   C Connecteur   Z Zone
  • D Dot-vote   G Grille   F Ajuster   T Timer
  • Suppr Supprimer   Échap Désélectionner   ? Aide

Interactions

  • 🖱️ Glisser fond = pan  ·  Molette = zoom
  • 🖱️ Double-clic = entrer dans l'élément  ·  Clic-droit → Modifier texte / menu
  • 🖱️ Shift+clic = multi-sélection  ·  Shift+drag = rectangle sélection
  • 🗳️ Vote : Shift+clic = retirer un vote  ·  📊 = résultats & budget
  • ⬜ Zones : double-clic = renommer  ·  glisser sticky → auto-cluster

Grist

  • 1 sticky = 1 ligne dans la table
  • Table auto-créée au 1er lancement ("Whiteboard_Elements")
  • Colonnes manquantes ajoutées automatiquement
  • Sélection synchronisée widget ↔ table
  • Multi-utilisateur via polling (3s)
